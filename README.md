# Hush

***

The reason I made this very simple chat app is to learn the basics of WebSockets. Of course that's not a good enough excuse to skip logging messages that users send, let's call that a security & privacy measure.

This project succeeded in its intention and for that I am satisfied. Personally I would never use this app myself, nowadays people are too fixated to their screens. Nothing beats good old social interaction in real life.

Applications are here to make things easier. Don't let them steal what makes you human.
